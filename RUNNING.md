# Running the Smart KOT System

This guide explains how to set up and run the Smart KOT system in different environments.

## Prerequisites
- **Node.js**: Version 22.x recommended.
- **MongoDB**: A local instance or MongoDB Atlas.

## 1. Environment Configuration
Create a `.env` file in the root directory (or ensure the following variables are set):

```env
MONGODB_URI=mongodb://your-mongodb-uri
PORT=8000
FRONTEND_URL=http://localhost:5173
RAZORPAY_KEY_ID=your_key_id
RAZORPAY_KEY_SECRET=your_key_secret
JWT_SECRET=your_jwt_secret
```

## 2. Development Mode
Run both the frontend and backend simultaneously for development.

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Start Frontend**:
    ```bash
    npm run dev
    ```
    The frontend will be available at `http://localhost:5173`.
3.  **Start Backend**:
    In a separate terminal:
    ```bash
    # Using the root script (requires build first or running source)
    npm run start
    ```
    *Note: For direct backend development, you can navigate to `backend/` and use a watcher like `nodemon` if installed.*

## 3. Production Mode (Manual Build)
1.  **Build the Project**:
    ```bash
    npm run build
    ```
    This builds the frontend into `dist/` and compiles the backend into `backend/dist/`.
2.  **Start the Server**:
    ```bash
    npm run start
    ```
    The server will run on `http://localhost:8000` and serve both the API and the React frontend.

## 4. Production Mode (Docker)
1.  **Build and Run**:
    ```bash
    docker build -t smart-kot .
    docker run -p 8000:8000 --env-file .env smart-kot
    ```
